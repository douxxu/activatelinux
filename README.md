# ActivateLinux
A windows like activation watermark, but for linux.

### Built with
* [![Nodejs][Node.js]][Node-url]
* [![ElectronJS][Electronjs]][Electron-url]




### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```


## Installation

Install ActivateLinux with
```sh
sudo npm install -g activatelinux
```

## Usage
Run it with 
```sh
activatelinux
```
